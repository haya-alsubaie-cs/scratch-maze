# Scratch Maze Game – KS2

## Overview
This project is an educational maze game created using Scratch.

The game is designed to introduce students to:
- Event-driven programming
- Coordinates
- Conditionals
- Collision detection
- Debugging strategies

## Learning Objectives
- Use event blocks to start a program
- Control sprite movement using x and y coordinates
- Apply conditional statements
- Detect collisions using color sensing

## Curriculum Alignment
UK KS2 Computing Curriculum:
- Design, write and debug programs
- Use logical reasoning to explain how algorithms work

## How to Play
Use arrow keys to move the sprite.
Avoid red walls.
Reach the red finish area to win.
